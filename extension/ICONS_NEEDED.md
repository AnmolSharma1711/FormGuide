# Icon Requirements

You need to create/add icons to your extension:

## Required Icons:
1. **16x16** - Favicon
2. **48x48** - Extension management page
3. **128x128** - Chrome Web Store

## Add to manifest.json:
```json
"icons": {
  "16": "icons/icon16.png",
  "48": "icons/icon48.png",
  "128": "icons/icon128.png"
}
```

## Design Tips:
- Simple, recognizable design
- Clear at small sizes
- Related to form/document/help theme
- Use your brand colors (currently gradient purple/blue)

## Free Tools:
- Canva.com - Easy icon creator
- Figma.com - Professional design
- favicon.io - Quick generator
